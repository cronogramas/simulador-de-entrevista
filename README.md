# Simulador de Entrevista Técnica

Este repositório documenta uma simulação de entrevista técnica conduzida para a vaga de **Desenvolvedor Júnior Java**.  
O objetivo é demonstrar como estruturar perguntas, respostas e análises em um formato de entrevista técnica.

---

## Estrutura da Entrevista

A entrevista foi conduzida em quatro etapas principais:

1. **Título da vaga e propósito**
2. **Senioridade esperada**
3. **Stack tecnológica**
4. **Soft Skills valorizados**

Após essas etapas, foi realizada uma simulação de entrevista com o candidato e, posteriormente, um resumo analítico.

---

## Perguntas e Respostas

### 1. Título da vaga e propósito
- **Pergunta:** Qual é o título da vaga e qual o propósito principal desse cargo?  
- **Resposta:** Dev Jr. Java

### 2. Senioridade
- **Pergunta:** Qual a senioridade esperada e por que?  
- **Resposta:** Junior

### 3. Stack tecnológica
- **Pergunta:** Quais tecnologias, frameworks e práticas são essenciais?  
- **Resposta:** Java, Log4j Framework e SQL

### 4. Soft Skills
- **Pergunta:** Quais comportamentos ou atitudes são mais valorizados?  
- **Resposta:** Comunicação e Transparência

---

## Simulação de Entrevista com o Candidato

### Experiência com Java
- **Pergunta:** Você poderia me contar sobre sua experiência prática com Java?  
- **Resposta:** Já usei para criar mods para Minecraft.

### Experiência com SQL
- **Pergunta:** Você já teve alguma experiência com bancos de dados SQL?  
- **Resposta:** Já criei um projeto de site que usava banco de dados SQL.

### Experiência com Log4j
- **Pergunta:** Você já trabalhou ou estudou o uso do Log4j para logging em aplicações Java?  
- **Resposta:** Não utilizei ainda, mas sei que pode ser usado para registrar logs úteis dentro de um servidor.

### Soft Skills
- **Pergunta:** Como você costuma demonstrar comunicação e transparência em projetos ou trabalhos em equipe?  
- **Resposta:** Geralmente faço meus projetos sozinho, mas me comunico com clareza e antecipo informações para evitar problemas.

### Objetivos de Carreira
- **Pergunta:** Quais são seus objetivos de carreira ao entrar como Dev Jr. Java?  
- **Resposta:** Quero ter mais conhecimento técnico, aprender mais sobre Java e evoluir profissionalmente.

## Plano de Estudos – 3h por dia

**Semana 1–2: Fundamentos de Java**  
- 1h30 por dia  
- Sintaxe, classes, objetos, herança, exceções.  
- Atividade: pequenos programas (calculadora, agenda).

**Semana 3–4: SQL**  
- 1h por dia  
- SELECT, INSERT, UPDATE, DELETE, JOINs.  
- Atividade: banco de dados para cadastro de usuários.

**Semana 5: Log4j**  
- 30min por dia  
- Configuração, níveis de log, boas práticas.  
- Atividade: implementar logs em projeto Java.

**Estrutura diária (3h):**  
- 1h30 Java  
- 1h SQL  
- 30min Log4j ou revisão
